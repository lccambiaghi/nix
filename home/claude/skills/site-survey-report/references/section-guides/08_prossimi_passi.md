# §8 — Prossimi Passi

## Key Questions Sections to Load
Next Steps, Post Survey

## Data Required
- Offer validity date
- Trial period (days)
- Installation duration (days)
- Decision maker names/roles from Q&A

## What To Write

A milestone-based implementation roadmap.

### Milestone 1: Revisione e Approvazione
- Review of technical report
- Technical decision maker (name + role from Q&A, or placeholder)
- Economic decision maker for budget approval
- Buying process requirements

### Milestone 2: Firma Contratto
- Contract signature
- First payment tranche (if applicable)
- Offer validity date

### Milestone 3: Preparazione Sito
- Power plug installation for gateways (if needed)
- Network configuration preparation
- Duration: 1-2 weeks

### Milestone 4: Installazione
- Sensor and gateway deployment
- Duration: N days
- System configuration and testing

### Milestone 5: Periodo di Prova (only if trial_days > 0)
- N-day evaluation period
- Baseline establishment
- Alert configuration tuning
- User training

### Milestone 6: Go-Live
- Full production monitoring begins
- Contract period starts
- Regular reporting cadence established

### Milestone 7: Scalabilità (only if scalability interest mentioned)
- Expansion potential to other sites

- Project Manager: Nicandro Buono (eMomentum)

- Executive sponsor: from Q&A or placeholder
- Weekly alignment with vibration expert
- Steering committee every two months
